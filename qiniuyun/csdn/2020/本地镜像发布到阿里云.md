## 镜像的生成方法

1、前面的DockerFile

2、从容器创建一个新的镜像  
 

```
docker commit [OPTIONS] 容器ID [REPOSITORY[:TAG]]
```

OPTIONS说明：  
\-a :提交的镜像作者；  
\-m :提交时的说明文字；
